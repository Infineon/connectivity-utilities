Defer Work to a Worker Thread.

Call the given function in the worker thread context. NOTE: If the thread priority is below that of the current thread, you will want to call osDelay( msec ) in your main thread.

Parameters
[in]worker_info: pointer to worker_thread used to run function
[in]work_func: function to run
[in]arg: opaque arg to be used in function call
Returns
success : 0
fail : 1
